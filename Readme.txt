WP Most Simple Social Bookmarks is a plug-in designed to add major social bookmarks to your website. 

== Description ==

This plugin is designed for beginners as well as expert users.  This plug-in, when installed, will display major social bookmarks after the post. All it needs is to upload and activate. No further configurations required at all. When installed, this plug-in will add 7 major bookmarking buttons after the end of post. These are twitter, delicious, digg, facebook, linked-in, yahoo & stumbleupon. 

If you want to change the bookmarking button image, simply replace the image in /images folder. If you don't want to touch the code at all, please maintain PNG as an extension of image files.



== Installation ==

This section describes how to install the plugin and get it working.



1. Upload `WP-Most-Simple-Social-Bookmarks.zip` to the `/wp-content/plugins/` directory. You can do this using 'Upload' functionality provided in plugins section of your wordpress dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are all set. Start bookmarking now

== Frequently Asked Questions ==

= How to add more bookmarks? =

As of now, adding another bookmark is done by editing the .php file of the plugin. If you are comfortable with simple php and bookmarking api, you can add additional <a> anchor tag in the existing file and save it. For format of the api, please refer to the respecitive social bookmarking website's api documentation
